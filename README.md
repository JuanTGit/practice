## 🎯 My Approach

This repository follows a **systematic 5-step problem-solving methodology**:

1. **Simplify the Problem**
   - Clarify inputs and expected outputs
   - Ask clarifying questions
   - Identify edge cases

2. **Pattern Recognition**
   - Identify the problem category
   - Find a straightforward solution first
   - Analyze time and space complexity

3. **Implementation Plan**
   - Outline the algorithm step-by-step

4. **Code**
   - Write clean, readable implementation

5. **Debug**
   - Fix syntax errors
   - Verify logic against test cases
   - Handle edge cases
